# 🗺️ Data Engineering Learning Roadmap

This roadmap outlines a structured path to becoming a proficient Data Engineer.

---

## Phase 1 – Programming Fundamentals

- SQL
- Python
- Git
- Linux

---

## Phase 2 – Data Engineering Basics

- ETL & ELT
- Data Modeling
- Data Warehousing
- Data Lakes
- File Formats (CSV, JSON, Parquet, ORC)

## Phase 3 – Big Data Technologies

- Apache Spark
- PySpark
- Kafka
- Hive

---

## Phase 4 – Cloud Data Engineering

### AWS

- S3
- Glue
- Athena
- Redshift
- Lambda
- DMS
- CloudWatch
- IAM

### Azure

- Azure Data Factory
- Azure Databricks
- ADLS Gen2
- Synapse Analytics
- Event Hubs
- Azure Monitor

---

## Phase 5 – Advanced Topics

- Streaming Pipelines
- CDC
- Delta Lake
- Medallion Architecture
- Performance Optimization
- Monitoring & Alerting
- Security
- Cost Optimization

---

## Phase 6 – Interview Preparation

- SQL
- PySpark
- AWS
- System Design
- Scenario-Based Questions

---

## Goal

Become a Production-Ready Data Engineer capable of designing scalable, reliable, and cost-effective data platforms.
